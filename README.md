# GitaGuide - Bhagavad Gita Wisdom for Every Problem

A web app that helps people find solutions to everyday life problems through the timeless wisdom of the Bhagavad Gita.

## Features
- 116 curated verses across 18 life themes
- Bilingual support (Hindi + English)
- Smart keyword matching engine
- Authentic translations (Swami Ramsukhdas for Hindi, Swami Gambirananda for English)
- Upvote/downvote feedback system
- Mobile-first responsive design

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

1. Push code to GitHub
2. Sign in at vercel.com with GitHub
3. Click "New Project", import your repo
4. Click "Deploy"
5. Live at yourproject.vercel.app

## Data Source
Verse data from the open-source gita/gita repository (github.com/gita/gita).
